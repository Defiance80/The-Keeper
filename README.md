# The Keeper — Recreation Operations & POS Command Platform

**Moody AFB 23 FSS Recreation Operations & POS Management**

The Keeper is a front-end demonstration prototype illustrating how a unified Recreation Management and POS platform could support Moody AFB 23 FSS operations. This prototype showcases a modern, responsive interface designed for recreation facility management, program registration, point-of-sale transactions, inventory tracking, financial reconciliation, and administrative oversight.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** TailwindCSS
- **State Management:** Zustand
- **Charts:** Recharts
- **Icons:** Lucide React
- **Deployment:** Vercel-ready
